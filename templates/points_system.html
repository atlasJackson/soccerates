Breaking Down the Points

Return to Form Hub

The Soccerates Points System comes in two different flavours: the Standard Points System is for a standard 90-minute game, awarding points based on the accuracy of the user's predictions when the match is finished. The Augmented Points System offers users a chance to gamble for additional points for matches which can go to Extra Time or to Penalties, typically found in the knockout stages of football tournaments. A full breakdown of these two systems can be found below.

Standard Points System

For fixtures that finish within 90 minutes.

Points for users' predictions are awarded based on the following criteria:

  • 5 points are awarded if the user's prediction is an exact match with the result.
  • 2 points are awarded if the user predicts the outcome of the match, i.e. selecting the correct winner or correctly predicting the match ending in a draw.
  • 1 point is awarded if the total goals in the user's prediction equals the total goals scored in the match.
  • 1 point is awarded if the goal difference in the user's prediction equals the goal difference between the two teams at the end of the match. This takes into account positive and negative goal difference: 2-0 is considered to be a goal difference of +2, whilst 0-2 is considered to be a goal difference of -2. Therefore, this point is only obtainable if the user also predicts the correct outcome.

Augmented Points System

For fixtures that go beyond 90 minutes.

For fixtures which can go over 90 minutes, there is the chance to gain points by selecting the option for the game going to Extra Time or Penalties. These points are awarded in addition to the Standard Points System. However, there are risks involved in selecting these options:

  • An additional 2 points will be awarded if the user correctly selects for the match going to Extra Time/Penalties.
  • If the user incorrectly selects these options, a 1 point penalty will apply.
  • These points are cumulative: if the user selects for the match to go to Extra Time and Penalties, 4 points will be awarded for a correct prediction; if the fixture goes to ET but not penalties, a net 1 point will be awarded. Finally, 2 points will be deducted for the game finishing in 90 minutes.

Whether or not the game goes to Extra Time/Penalties, your prediction will be matched against the final result.

  • No points will be awarded/deducted if both options are ignored.
  • There are no predictions in place for the outcome of a Penalty Shootout (if it applies), only for whether or not the game reaches that stage.
