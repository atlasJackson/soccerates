Breaking Down the Points

  • Points are awarded based on the accuracy of your prediction, with a maximum of 5 points awarded for a perfect match. You can find a summary of the points system in the graphic at the bottom of the page.
  • Use the arrows to increment/decrement the number shown in the box, or use the keyboard to manually enter a number into each box.
    • {% if stage_is_group %}
  • Use the tabs at the top to display the fixtures for each group.
    • {% endif %}
  • Clicking submit will submit your entire set of predictions - there is no need to submit each prediction individually.
  • Submission for a fixture will be disabled 15 minutes prior to kick-off.
  • For fixtures which can go over 90 minutes, there is an additional points system:
    • An additional 2 points will be awarded if the user correctly selects for the match going to Extra Time/Penalties.
    • If the user incorrectly selects these options, a 1 point penalty will apply.
    • These points are cumulative: if the user selects for the match to go to Extra Time and Penalties, 4 points will be awarded for a correct prediction; if the fixture goes to ET but not penalties, a net 1 point will be awarded. Finally, 2 points will be deducted for the game finishing in 90 minutes.
    • No points will be awarded/deducted if both options are ignored.
    • Your prediction will be matched against the result after Extra Time if the game is tied at the end of the standard 90 minutes.
